Логика на завтра

Главное — Codex CLI. Подписка ChatGPT Business, которая у вас уже оплачена, открывает Codex CLI без дополнительных ключей и API-кредитов. Логин в одно нажатие через браузер.

Удобство — Handy

Зажал клавишу — продиктовал — отпустил, текст вставляется в то поле, где курсор. В том числе в Codex. Распознаёт локально, аудио не уходит в облако — уместно с вашими NDA. Ставится за пару минут на Mac и Windows.

Claude Code — опционально. Регистрация в Claude.ai через российский номер не идёт. Один из путей — eSIM Казахстана/Сербии. Это второстепенная задача: если у Рода до утра получится завести Anthropic — поставим Claude Code сверху. Если нет — занятие пройдёт целиком на Codex.

Минимум к утру: у Рода и у Дмитрия в терминале запускается команда codex и открывает интерфейс. Всё остальное — бонус.

Что такое Codex CLI и зачем

Codex — CLI-агент от OpenAI, живёт в терминале, читает файлы на вашем компьютере, исполняет команды и пишет код прямо в проект. В отличие от чата ChatGPT в браузере — он работает в среде вашей машины, видит вашу файловую систему, понимает что у вас открыто, создаёт и правит реальные файлы.

Это не «чат с улучшениями», это другой класс инструмента: вы перестаёте копировать туда-сюда, начинаете давать задачи и получать готовый результат — файл, страницу, скрипт, репозиторий.

На занятии разберём

Цель — пройти полную цепочку от запроса до опубликованного результата своими руками. У каждого получится готовый артефакт в вашем GitHub.

  • Скиллы — переиспользуемые рецепты, которые вы один раз описали словами и дальше вызываете одной командой
  • Агенты — делегирование задачи отдельному «специалисту» с собственным контекстом (ресёрчер, копирайтер, дизайнер, ревьюер)
  • MCP — подключаемые внешние инструменты: поиск, доступ к файлам, к GitHub, к браузеру, к вашим локальным сервисам
  • End-to-end pipeline — пройдём всю цепочку: поиск информации → обработка → сборка сайта → публикация в GitHub Pages под вашим аккаунтом

В конце занятия у каждого из вас будет: установленный Codex, понимание трёх ключевых концепций (скилл / агент / MCP) и собственная страница в интернете, которую вы собрали голосом и руками за полтора часа.

Ответы на вопросы из вчерашних голосовых

Если Claude не получится поставить — критично?

Нет, не критично. Занятие построено на Codex. Claude Code — это дополнительный инструмент, который мы поставим, если у Рода удастся завершить регистрацию (через зарубежную eSIM или сервис временных номеров). До утра тратить силы на Claude не нужно — лучше довести Codex до запуска у обоих.

Личные или корпоративные аккаунты — что выбрать сейчас

На стадии «двое за две недели» — личные. Корпоративная схема с расчётным счётом для зарубежных подписок не собирается без отдельной юр-конструкции, и с учётом ваших NDA и регуляторных ограничений переход на enterprise — отдельный спринт после того, как два человека реально начнут получать пользу. Сейчас корректно: личные аккаунты Рода и Дмитрия, оплата с зарубежной карты владельца или айтишника.

Что с NDA и чувствительной документацией

Любые документы под NDA — ни в Codex, ни в Claude, ни в ChatGPT в открытом виде не отправляем. Тренируемся на обезличенных или публичных материалах — задачи и принципы те же, а данные ничьих NDA не нарушают. Под чувствительный класс (внутренние документы, проекты под регуляторные ограничения) уместен отдельный контур на локальных моделях — это материал следующей встречи, не сегодня.

Род · macOS

Полный путь — около 15 минут на чистом маке. Если Node.js уже стоит — 5 минут.

Шаг 1Открыть Терминал

На Mac: ⌘ + пробел → набрать Terminal → Enter. Откроется чёрное (или белое) окно. Это и есть терминал, в нём всё происходит.

Шаг 2Поставить Node.js

Codex CLI распространяется через npm — менеджер пакетов из мира Node.js. Поэтому сначала Node.

Самый простой путь: зайти на nodejs.org, нажать большую зелёную кнопку «Download Node.js (LTS)», открыть скачанный .pkg, нажимать Continue / Install. Установщик стандартный.

После установки в терминале проверить:

node -v
npm -v

Должны вывестись две строчки с номерами версий, например v22.11.0 и 10.9.0. Если вышли — идём дальше.

command not found: node

Закройте терминал и откройте заново — установщик Node иногда не сразу подцепляется в открытую сессию. Если и после этого не работает — пришлите скриншот в Telegram @timzinin.

Шаг 3Поставить Codex CLI

npm install -g @openai/codex

Установка занимает 30–60 секунд. В конце должно появиться сообщение added N packages.

Проверить:

codex --version
EACCES / Permission denied на npm install -g

Запустите ту же команду с sudo:

sudo npm install -g @openai/codex

Терминал спросит пароль от мака — это пароль учётной записи macOS, не от ChatGPT.

Шаг 4Войти в свой ChatGPT-аккаунт

В терминале:

codex

При первом запуске Codex предложит выбрать способ авторизации. Выбираем Sign in with ChatGPT — Codex откроет браузер. В браузере залогиниться той же почтой, что и ваш ChatGPT Business аккаунт за 21 €. Подтвердить разрешение для CLI. Браузер скажет «You can close this tab». В терминале появится приглашение Codex.

Подписка Business ChatGPT & Codex уже включает доступ к Codex CLI без отдельных API-ключей. Дополнительно ничего платить не нужно.

Шаг 5Проверка боем

В Codex (после логина) набрать на русском:

напиши маленький python-скрипт, который печатает "сила тока" и сегодняшнюю дату

Codex покажет план, спросит подтверждение и создаст файл рядом. Если файл создался — установка прошла. Можно закрывать терминал — занятие проведём с этого же места.

Шаг 6Handy — голосовой ввод в терминал

бесплатноopen sourceлокально

Speak into any text field

Маленькая утилита, которая превращает Mac в надиктовщика: зажал клавишу → говоришь → отпустил, текст вставляется в курсор. В том числе в окно Codex.

Зачем для нашей задачи: ставить задачу Codex, описывать что вы хотите получить на выходе, надиктовывать длинные тексты и заметки — намного быстрее, чем набирать. Распознавание происходит локально на маке, ничего не уходит в облако.

Скачать .dmg под ваш мак с handy.computer/download:

  • Apple Silicon (M1/M2/M3/M4 — почти все маки с 2020 года) → Handy_0.8.3_aarch64.dmg
  • Intel (старые маки до 2020) → Handy_0.8.3_x64.dmg

Узнать какой у вас процессор: яблоко в углу → About This Mac. Если в строке Chip написано Apple M1/M2/M3/M4 — берёте Apple Silicon. Если Intel — Intel.

Установка стандартная: открыть .dmg, перетащить иконку Handy в папку Applications. При первом запуске macOS попросит разрешить доступ к микрофону и accessibility (нужно чтобы вставлять текст в чужие приложения) — System Settings → Privacy & Security разрешить.

Горячая клавиша по умолчанию настраивается при первом запуске. Возьмите что-то редкое — например F5 или Right Option. Дальше: зажали → продиктовали запрос → отпустили → текст вылезет в Codex.

ОпциональноClaude Code

Делать только если уже разобрались с регистрацией Anthropic (eSIM Казахстана/Сербии). Если регистрация в Claude.ai не пройдена — пропускайте, утром достаточно одного Codex.

npm install -g @anthropic-ai/claude-code

Запуск:

claude

Откроется браузер для логина в Claude.ai. После логина в терминале появится приглашение Claude Code. На макс/про подписку он работает «из коробки».

Регистрация Anthropic — почему телефон отказывает

Anthropic не принимает российские и белорусские номера на этапе верификации. Рабочие пути: eSIM Казахстана (например, KCell), eSIM Сербии (через знакомых на месте, без паспорта), временный номер 5sim.net (риск — номер может быть «сожжён» прошлыми попытками регистрации в Anthropic). Любой иностранный реальный номер сработает.

Дмитрий · Windows

Полный путь — около 20 минут на чистой винде. Никакого WSL, никакого Linux — всё работает в обычной PowerShell.

Шаг 1Открыть PowerShell

Нажать Win + X → выбрать Terminal или PowerShell. Откроется синее окно с приглашением. Если Terminal в меню нет — наберите в поиске Windows слово «PowerShell» и откройте его.

Все команды ниже копируются в это окно (правой кнопкой мыши вставляется текст).

Шаг 2Поставить Node.js

Codex CLI распространяется через npm — менеджер пакетов Node.js. Поэтому сначала Node.

Зайти на nodejs.org, нажать большую зелёную кнопку «Download Node.js (LTS)», открыть .msi-файл, нажимать Next → Install → Finish. Установщик стандартный, ничего не меняем.

После установки закрыть PowerShell и открыть заново (это важно — иначе он не увидит новый Node). Проверить:

node -v
npm -v

Должны вывестись две строчки с номерами версий, например v22.11.0 и 10.9.0.

Шаг 3Поставить Codex CLI

npm install -g @openai/codex

Установка идёт 30–60 секунд. В конце появится added N packages.

Проверка:

codex --version
«не удаётся выполнить сценарий, выполнение сценариев отключено»

Windows по умолчанию блокирует запуск глобально установленных скриптов. Один раз разрешаем для текущего пользователя:

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

На вопрос ответить Y. После этого codex --version сработает.

«codex не является внутренней или внешней командой»

Закройте PowerShell и откройте заново. Если не помогло — перезагрузите компьютер один раз (Windows иногда долго прописывает PATH).

Шаг 4Войти в свой ChatGPT-аккаунт

В PowerShell:

codex

При первом запуске Codex предложит выбрать способ авторизации. Выбираем Sign in with ChatGPT — Codex откроет браузер. В браузере залогиниться той же почтой, что и ваш ChatGPT Business аккаунт за 21 €. Подтвердить разрешение. Браузер скажет «You can close this tab». В PowerShell появится приглашение Codex.

Подписка Business ChatGPT & Codex уже включает доступ к Codex CLI без отдельных API-ключей. Ключи и кредиты OpenAI отдельно покупать не нужно.

напиши маленький python-скрипт, который печатает "сила тока" и сегодняшнюю дату

Speak into any text field

Маленькая утилита, которая превращает Windows в надиктовщика: зажал клавишу → говоришь → отпустил, текст вставляется в курсор. В том числе в PowerShell с Codex.

Скачать установщик с handy.computer/download:

  • Обычный ПК или ноутбук с Intel/AMD (99% случаев) → Handy_0.8.3_x64-setup.exe
  • Windows on ARM (если у вас Surface Pro X или похожий ARM-ноутбук) → Handy_0.8.3_arm64-setup.exe

Узнать какой процессор: правой кнопкой по «Этот компьютер» → Свойства. Если процессор Intel или AMD — берёте x64. Если Snapdragon/ARM — arm64.

Установка стандартная: запустить .exe, нажать Next → Install → Finish. Windows может предупредить «не известный издатель» — More info → Run anyway. При первом запуске разрешить доступ к микрофону.

Делать только если получится зарегистрировать аккаунт в Claude.ai. На винде Claude Code работает нативно через ту же PowerShell — WSL2 не нужен.

npm install -g @anthropic-ai/claude-code

Запуск:

claude

Откроется браузер для логина в Claude.ai. На вашу проблему с телефоном смотрите ниже.

«Claude.ai требует заграничный номер» — пути решения

Anthropic не принимает российские и белорусские номера. Варианты по убыванию надёжности:

  • eSIM Казахстана — оператор KCell, оформляется онлайн, есть постоянный номер для будущих верификаций
  • eSIM Сербии — через знакомых на месте, паспорт не требуется
  • 5sim.net — временный номер. Дёшево, но номер мог быть «сожжён» предыдущими регистрациями в Anthropic, гарантий нет

Если до утра не получится — это нормально. Утром поставим только Codex и начнём занятие.
